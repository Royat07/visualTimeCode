# Visual Metronome Enhanced

一个功能丰富的轻量级视觉节拍器，通过屏幕闪烁提供节拍提示。

## 功能特性

### 基础功能
- **可配置参数**：闪烁时间、闪烁间隔、批次大小、批次间隔
- **多种颜色格式**：支持 HEX、RGB、HSL 颜色格式
- **实时状态显示**：左上角显示当前设置信息
- **纯视觉实现**：无声音元素，仅通过屏幕闪烁工作

### 增强功能
- **统一控制按钮**：Start/Stop 合并为一个按钮，根据状态自动切换
- **启动即显示设置**：程序启动时自动打开设置面板
- **颜色预设**：提供常用颜色快速选择
- **HSL 滑块控制**：专用的HSL滑块界面，实时预览
- **预设管理**：保存、加载、删除自定义预设
- **预设导入/导出**：支持 JSON 格式的预设文件
- **配置文件支持**：通过 config.json 设置默认预设

### Bug 修复 (v2.0)
- ✅ **修复预设保存问题**：保存预设时现在正确读取当前输入框的值，而不是旧的状态值
- ✅ **修复 RGB 颜色选择**：添加了专用的 RGB 数字输入界面，支持红、绿、蓝三个通道独立调节
- ✅ **修复 HEX 颜色选择**：保持原有的 HEX 文本输入和颜色选择器
- ✅ **修复 HSL 颜色选择**：添加了专用的 HSL 滑块控制界面，支持实时预览
- ✅ **修复默认预设加载**：config.json 中的预设现在正确加载，支持默认预设自动应用
- ✅ **添加设置默认预设功能**：可以将任意保存的预设设为默认预设
- ✅ **添加配置文件导出功能**：可以导出包含默认预设的完整配置文件

## 使用说明

### 基本操作

1. **启动程序**：打开 `Visual Metronome Enhanced.html`
2. **配置设置**：启动时会自动显示设置面板
3. **开始节拍**：点击 "▶ Start" 按钮或按空格键
4. **停止节拍**：点击 "⏹ Stop" 按钮或再次按空格键
5. **修改设置**：点击 "⚙️ Settings" 按钮打开设置面板

### 设置参数

- **Flash Duration**：每次闪烁的持续时间（秒）
- **Flash Interval**：两次闪烁之间的时间间隔（秒）
- **Batch Size**：每轮闪烁的次数
- **Batch Interval**：批次之间的休息时间（秒）

**示例**：设置 Flash Duration=0.1s, Flash Interval=1s, Batch Size=3, Batch Interval=3s
- 屏幕将每隔 1 秒闪烁一次，每次持续 0.1 秒
- 闪烁 3 次后，间隔 3 秒开始下一轮
- 无限循环，直到手动停止

### 颜色设置

1. 选择颜色格式（HEX/RGB/HSL）
2. 从预设色块中快速选择常用颜色
3. 或使用颜色选择器自定义颜色
4. 支持分别设置闪烁颜色和背景颜色

### 预设管理

#### 保存预设
1. 配置好所有参数
2. 点击 "💾 Save Preset" 按钮
3. 输入预设名称并确认

#### 加载预设
1. 点击 "📁 Load Preset" 按钮
2. 在预设列表中点击要加载的预设
3. 设置会自动应用

#### 删除预设
1. 打开预设列表
2. 点击预设右侧的 🗑️ 按钮
3. 确认删除

#### 导出预设
1. 配置好参数或选择已保存的预设
2. 点击 "📤 Export" 按钮
3. 预设会以 JSON 文件格式下载

#### 导入预设
1. 点击 "📥 Import" 按钮
2. 选择要导入的 JSON 文件
3. 预设会自动添加到预设列表

## 配置文件

创建 `config.json` 文件来自定义默认行为：

```json
{
  "defaultPreset": "Standard 1-second beat",
  "autoStart": false,
  "description": "Visual Metronome Configuration File",
  "presets": [
    {
      "name": "Standard 1-second beat",
      "description": "Classic 1-second interval",
      "settings": {
        "flashDuration": 0.1,
        "flashInterval": 1,
        "batchSize": 3,
        "batchInterval": 3,
        "flashColor": "#ff0000",
        "bgColor": "#0a0a0a"
      }
    }
  ]
}
```

### 配置选项

- **defaultPreset**：启动时自动加载的预设名称
- **autoStart**：是否自动开始节拍（默认 false）
- **presets**：预定义的预设列表

## 快捷键

- **空格键**：启动/停止节拍器
- **ESC**：关闭设置面板或对话框

## 预设存储

- 预设保存在浏览器的 localStorage 中
- 清除浏览器数据会删除所有保存的预设
- 建议定期导出重要预设进行备份

## 故障排除

### 预设无法加载
- 检查 localStorage 是否被禁用
- 确认 JSON 文件格式正确
- 尝试清除浏览器缓存后重新加载

### 闪烁不正常
- 检查浏览器性能设置
- 关闭其他占用资源的应用
- 尝试降低闪烁频率

### 配置文件不生效
- 确认 config.json 与 HTML 文件在同一目录
- 检查 JSON 格式是否正确
- 尝试硬刷新浏览器（Ctrl+F5）
